# UAE Smart Meter Demand & Supply Analysis 2023

## Project Overview
SQL analysis of UAE smart meter data analyzing 
consumption patterns across residential, commercial 
and industrial consumers in Dubai, Abu Dhabi 
and Sharjah for 2023.

## Business Questions Answered
1. Total consumption by consumer type
2. Average monthly bill by region
3. Top 3 highest consuming meters
4. Peak vs off-peak usage ratio
5. Meters above average consumption
6. Monthly industrial consumption trend
7. Total revenue by region
8. Month over month consumption change

## Key Findings
- Industrial = 67% of total consumption
- M003 Dubai Industrial peaked at 13,500 kWh in July
- Peak hour usage = 50% for Industrial and Commercial
- Dubai highest revenue region due to Industrial consumer
- July peak month across all types — UAE summer effect
- M003 only meter above overall average consumption

## SQL Concepts Used
- SELECT, FROM, WHERE, GROUP BY
- HAVING with Subquery
- ORDER BY, LIMIT
- Window Functions — LAG
- ROUND, COUNT DISTINCT
- Aggregations — SUM, AVG
